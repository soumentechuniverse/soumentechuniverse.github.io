Module 1: Computer Basics & History

What is a Computer?

A computer is an advanced electronic device that takes raw data as input from the user and processes it under the control of a set of instructions (called program), produces a result (output), and saves it for future use.

Brief History & Origin

The history of the computer dates back to the invention of the Abacus in Babylonia around 2400 BC. However, the foundation of modern computing was laid by Charles Babbage, who designed the Analytical Engine in 1837. He is rightfully known as the "Father of the Computer".

Generations of Computers

  • 1st Gen (1940-1956): Vacuum Tubes (e.g., ENIAC). Very large, expensive, and emitted enormous heat.
  • 2nd Gen (1956-1963): Transistors. Smaller, faster, and more energy-efficient.
  • 3rd Gen (1964-1971): Integrated Circuits (ICs). Introduction of keyboards, monitors, and early operating systems.
  • 4th Gen (1971-Present): Microprocessors (VLSI/ULSI circuits). Brought computing to desktop setups and personal smartphones.

Module 2: Advanced Networking

Introduction to Computer Networks

Data communication and computer networks form the backbone of modern cyber space. It allows multiple computer nodes to share data resources seamlessly using wired or wireless media protocols.

The OSI Model (7 Layers)

The Open Systems Interconnection model defines a conceptual framework for network communications:

  1. Physical Layer: Transmission of raw bit streams over physical media.
  2. Data Link Layer: Node-to-node data transfer and framing (MAC addresses).
  3. Network Layer: Routing paths and logical addressing (IP addresses).
  4. Transport Layer: Reliable end-to-end connection (TCP/UDP protocols).
  5. Session Layer: Managing sessions between applications.
  6. Presentation Layer: Data translation, encryption, and formatting.
  7. Application Layer: Human-computer interaction (HTTP, FTP, SMTP).

Module 4: Practical Cyber Security

The CIA Triad

The core objective of security is mapped using three main parameters:

  • Confidentiality: Preventing unauthorized access to data.
  • Integrity: Ensuring information cannot be modified unnoticed.
  • Availability: Making sure verified users have uninterrupted system access.
